onchainOS CLI - interact with OKX Web3 backend

Usage: onchainos.exe [OPTIONS] <COMMAND>

Commands:
  market       Market data: prices, charts, wallet PnL
  signal       Smart money / whale / KOL signal tracking
  memepump     Meme / pump.fun token scanning and analysis
  leaderboard  Leaderboard (top traders ranked by PnL, win rate, or volume)
  token        Token information
  swap         DEX swap
  gateway      On-chain gateway
  portfolio    Wallet portfolio and balances
  mcp          Start as MCP server (JSON-RPC 2.0 over stdio)
  wallet       Agentic wallet: login, verify, create, switch, status, logout, balance
  security     Security scanning (tx-scan, token-scan, dapp-scan, sig-scan)
  payment      Payment protocols ΓÇö auto-pay gated APIs (x402, etc.)
  upgrade      Upgrade onchainos to the latest version
  help         Print this message or the help of the given subcommand(s)

Options:
      --base-url <BASE_URL>  Backend service URL (overrides config)
      --chain <CHAIN>        Chain: ethereum, solana, base, bsc, polygon, arbitrum, sui, etc
  -h, --help                 Print help
  -V, --version              Print version
